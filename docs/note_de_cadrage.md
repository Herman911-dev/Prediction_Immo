# Note de Cadrage - Projet Prediction Immo

## 1. Contexte du projet
Le projet vise à exploiter les données DVF (2021-2025) pour fournir une aide à l'estimation immobilière fiable.

## 2. Objectifs techniques
* Ingestion massive de données foncières (Open Data).
* Entraînement d'un modèle de Machine Learning (Regression).
* Interface utilisateur interactive (Streamlit).

## 3. Résultats attendus
* Un dashboard permettant de saisir les caractéristiques d'un bien.
* Une prédiction de prix basée sur l'historique quinquennal.

## 4. Normes et Qualité
* Code respectant la norme **PEP8**.
* Documentation technique (> 10%).
* Accessibilité numérique conforme au **RGAA**.
